## @dylan/eslint-config

### Installation

Install with `npm` (or `yarn`):

```bash
npm i @dylan/eslint-config --save-dev
```

Then create an `.eslintrc` file in your project directory:

```json
{
  "extends": "@dylan"
}
```

### Configuring

Adjust your `.eslintrc` file as necessary by adding rules:

```json
{
  "extends": "@dylan",
  "rules": {
    "no-tabs": "off"
  }
}
```
